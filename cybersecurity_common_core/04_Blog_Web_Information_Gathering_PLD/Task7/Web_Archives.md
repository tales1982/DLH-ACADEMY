# Wayback Machine - Quick Reference

> The **Wayback Machine** (by the Internet Archive) is a digital archive of the web that has been capturing website snapshots since **1996**. It lets you "travel back in time" and view how a website looked and functioned at any point in its history — design, content, files, and all.

---

## How It Works

Web crawlers automatically snapshot websites at regular intervals, storing the **full page content** HTML, CSS, JavaScript, images, and other resources not just metadata.

---

## Why It Matters for Recon

| Use Case | Description |
|---|---|
| **Uncover hidden assets** | Old pages, directories, files, subdomains no longer on the live site |
| **Find exposed secrets** | API keys, credentials, or config files that were briefly public |
| **Track changes** | Compare snapshots over time to spot structural or tech changes |
| **Identify old vulnerabilities** | Outdated software versions or misconfigs from a previous era |
| **OSINT gathering** | Past employees, marketing pages, press releases, tech stack history |
| **Stealthy recon** | Purely passive, you never touch the live target |
