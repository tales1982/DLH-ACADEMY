# Understanding Vulnerabilities

Educational article series on cybersecurity vulnerabilities — from the foundational concept through attack and defense techniques, with a focus on practical understanding and real-world application.

---

## Contents

| File | Topic |
|------|-------|
| [task-1.md](task-1.md) | **Introduction to Vulnerabilities** — concept, historical context (Morris Worm, Heartbleed, EternalBlue), types (software, hardware, network, human), and organizational impact |
| [task-2.md](task-2.md) | **Static and Dynamic Analysis Tools** — difference between static analysis (SonarQube, Semgrep, Checkmarx) and dynamic analysis (Burp Suite, OWASP ZAP, Nessus); injection attacks (SQL Injection, XSS, Command Injection, LDAP Injection) |
| [task-3.md](task-3.md) | **Analysis Tools (continued)** — deeper look at static and dynamic analysis and injection attack prevention |
| [task-4.md](task-4.md) | **CSRF — Cross-Site Request Forgery** — how the attack works, its impact, and mitigation strategies (anti-CSRF tokens, origin validation, SameSite/Secure/HttpOnly cookie attributes) |
| [task-5.md](task-5.md) | **Importance of Patches and Updates** — why keeping systems updated is critical; real-world cases (WannaCry 2017, Equifax 2017); patch management automation |

## Key Topics

- Types of vulnerabilities: software, hardware, network, and human
- SAST (Static Application Security Testing) and DAST (Dynamic Application Security Testing) tools
- Injection attacks: SQL Injection, XSS, Command Injection, LDAP Injection
- Cross-Site Request Forgery (CSRF) and mitigation techniques
- Patch management and vulnerability lifecycle
