## Key Concepts

- **CVE** (Common Vulnerabilities and Exposures): universal vulnerability identification system
- **CWE** (Common Weakness Enumeration): catalog of software and hardware weakness types
- **NVD** (National Vulnerability Database): NIST database that enriches CVEs with CVSS scores
- **CVSS** (Common Vulnerability Scoring System): standardized severity metric (0.0 to 10.0)
