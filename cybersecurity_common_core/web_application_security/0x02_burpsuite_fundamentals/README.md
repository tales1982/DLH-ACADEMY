## Burp Suite Modules Covered

| Module | Purpose |
|--------|---------|
| **Proxy** | Intercepts HTTP/S traffic between browser and server |
| **Repeater** | Manually resends and modifies individual requests |
| **Intruder** | Automates payloads for fuzzing and brute-force attacks |
| **Scanner** | Actively scans for common web vulnerabilities (Pro only) |
| **Decoder** | Encodes and decodes data (URL, Base64, HTML, etc.) |
| **Comparer** | Compares two requests or responses |

---

## Skills Practiced

- Setting up Burp Suite as an HTTP proxy with browser configuration
- Intercepting and modifying HTTP requests and responses
- Using Repeater to test different payloads on an endpoint
- Understanding HTTP request structure (headers, cookies, body, parameters)
- Identifying vulnerabilities through manual request manipulation
- Capturing and analyzing authentication flows
