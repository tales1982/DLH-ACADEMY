# 0x00 — Introduction to Cybersecurity

Introductory cybersecurity module focused on Linux command-line fundamentals and system security basics — essential skills for any security professional.

---

## Contents

| File | Description |
|------|-------------|
| `0-release.sh` | Displays operating system release information |
| `1-gen_password.sh` | Generates a random password using system tools |
| `2-sha256_validator.sh` | Validates file integrity using a SHA-256 hash |
| `3-gen_key.sh` | Generates cryptographic keys for security use |
| `4-root_process.sh` | Identifies and lists processes running with root privileges |

---

## Skills Practiced

- Basic Linux shell command usage
- Generating and validating cryptographic hashes
- Identifying privileged processes on the system
- Concepts of authentication and data integrity
