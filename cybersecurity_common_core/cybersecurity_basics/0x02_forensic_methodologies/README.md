# 0x02 — Forensic Methodologies

Digital forensics module covering the investigation of security incidents through evidence analysis, intruder identification, and attack comprehension.

---

## Contents

| File | Description |
|------|-------------|
| `0-mystery.txt` | Initial investigation — starting point of the forensic scenario |
| `1-location.txt` | Identification of the threat's location within the system |
| `2-intruder_intent.txt` | Analysis of the intruder's intent based on the evidence |
| `3-vulnerability_type.txt` | Classification of the vulnerability type that was exploited |
| `4-attack_method.txt` | Identification of the attack method used |
| `5-indirect_impact.txt` | Analysis of the indirect impact of the incident |
| `5-ripple_effect.txt` | Ripple effect — how the incident affected other systems |

---

## Skills Practiced

- Digital forensic evidence analysis
- Identification of Indicators of Compromise (IoC)
- Determining attack vectors and techniques used
- Security incident documentation and reporting
- Understanding the impact chain of an attack (direct and indirect)
