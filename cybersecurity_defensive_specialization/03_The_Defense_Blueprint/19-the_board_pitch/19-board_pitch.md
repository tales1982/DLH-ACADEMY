# Board Pitch — MedDefense Health Systems

**To:** The Board of Directors
**For:** Monday's Meeting — Security Budget Approval

**Current State:** Four weeks of assessment found an organization with no security framework, no way to detect an attack in progress, and a network where every system can reach every other system. The verdict: we are not unlucky — we are unprotected, and it has already cost us twice.

**The Risk:** Our own data shows a ransomware attack that encrypts patient records and locks down billing carries an expected annual cost of $300,000 — not a worst case, the actuarially expected cost of leaving today's gaps open. That number is built from a threat pattern that has already hit us once, through a door that is still unlocked.

**The Plan:** We recommend investing $103,400 of the approved $120,000 in five specific fixes, not a general "more security" purchase. The top three: multi-factor authentication using software we already own ($4,000), a monitoring system that finally lets us detect an attack instead of discovering it by accident weeks later ($28,000), and closing one confirmed, actively-exploited flaw on our patient-records server ($500). Together with two additional fixes — protected backups and a real perimeter at our Westside clinic — this plan closes over half a million dollars in identified annual risk, verified with the same rigor a financial audit would demand.

**The Return:** For every dollar spent, we avoid approximately $5.68 in expected annual loss — a return few other investments this Board reviews this year will match, because unlike most spending decisions, the cost of *not* acting is not hypothetical. It is a number built from what already happened to us.
